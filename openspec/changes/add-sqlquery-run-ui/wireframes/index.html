$sqlquery-run UI wireframes

Lo-fi wireframes for the new "SQL query" mode of the SQL on FHIR page in the Pathling admin UI.

  • 1. SQL query, inline (Provide Library tab)

    Authoring a SQLQuery Library inline: SQL text, related ViewDefinitions, declared parameters, runtime values and output options.

  • 2. SQL query, stored (Select Library tab)

    Selecting a stored SQLQuery Library, previewing its SQL and binding runtime parameter values.

  • 3. Result card (tabular)

    Successful query rendered as a Table with format badge, row count and download.

  • 4. Result card (error)

    Server-side validation error surfaced in a red Callout next to the offending SQL.

Lo-fi monochrome - layout and information density only, not visual design.
